# Curious Crystals

**Status:** Archived (Completed Academic Project)
**Minecraft Version:** 1.20.1
**Platform:** Fabric

Curious Crystals was a collaborative modding project developed for Informatica Project 1. The goal was to expand Minecraft with new magical elements, including gems, mobs, structures, and gameplay mechanics. This repository represents the final submission of the course.

## Features

### Gem Upgrade System
Players can upgrade weapons and armor using a Smithing Table, a Gem Upgrade Template, and specific gems. Each gem grants unique abilities based on its origin dimension.

#### Overworld Gems
- **Sapphire**: Grants extra hearts.
- **Jade**: Applies random positive potion effects on hit.
- **Citrine**: Increases saturation restoration.
- **Opal**: Provides regeneration over time.

#### Nether Gems
- **Ruby**: Enables life steal on attack.
- **Peridot**: Chance to shock enemies on hit.
- **Garnet**: Adds extra damage and fire resistance upon hitting an enemy.
- **Alexandrite**: Acts as a Totem of Undying and adds knockback.

#### End Gems
- **Onyx**: Grants Speed and Strength buffs after killing an entity.
- **Bismuth**: Teleports the player back to spawn upon death or activation.

### Structures
New generated structures across dimensions:
- **Overworld**: Amethyst Arachnid Den, Crystal Cave, Explorer’s Hut.
- **Nether**: Piglin Gem Vault, Nether Dungeon.
- **End**: Fractured Geode, Gem Temple.

### Mobs
- **Gem Arachnid**: A hostile mob found in crystal caves, dropping rare gems. Designed by [Naam Product Owner].

## Installation

This project is archived and no longer receiving updates. To run it locally:

1. Install Fabric Loader for Minecraft 1.20.1.
2. Place `curious-crystals.jar` in your mods folder.
3. Launch the game.
