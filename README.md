## <span style="color:#00eaff;">New Gems & Upgrades</span>

Each gem grants a unique ability when combined with an item using the **Smithing table**  
(for example: diamond sword + ruby + gem upgrade template = ruby sword)

### Weapons & Abilities
- <span style="color:#b95cff;"><b>Gem Upgrade Template</b></span> — Upgrades your weapon/armor  
**Overworld**
- <span style="color:#66a3ff;"><b>Sapphire</b></span> — Extra Hearts  
- <span style="color:#6aff6a;"><b>Jade</b></span> — Random Positive Potion Effect on Hit  
- <span style="color:#ffe066;"><b>Citrine</b></span> — More Saturation  
- <span style="color:#b95cff;"><b>Opal</b></span> — Regeneration

**Nether**
- <span style="color:#ff4d4d;"><b>Ruby</b></span> — Life Steal  
- <span style="color:#6aff6a;"><b>Peridot</b></span> — Chance to Shock enemies on Hit  
- <span style="color:#ff4d4d;"><b>Garnet</b></span> — Extra Damage + Fire Resistance on Hit  
- <span style="color:#c77bff;"><b>Alexandrite</b></span> — Totem Effect + Knockback  

**End**
- <span style="color:#323232;"><b>Onyx</b></span> — Speed & Strength on Kill  
- <span style="color:#d4a373;"><b>Bismuth</b></span> — Return to Spawn 

### Gem Progression
- <span style="color:#00cc44;">Overworld:</span> basic gems  
- <span style="color:#ff6b6b;">Nether:</span> stronger gems  
- <span style="color:#9d4edd;">End:</span> top-tier gems

## <span style="color:#ffb84d;">Smithing table</span>

How it works:

1. Place a base item (e.g. <b>Diamond Sword</b>)  
2. Add a gem
3. Add the gem upgrade template
4. Receive your <b>upgraded</b> version with special abilities  

---

## <span style="color:#8be9fd;">Installation</span>

1. Install **Minecraft 1.20.1**  
2. Install **<a href="https://fabricmc.net/" target="_blank">Fabric Loader</a>**  
3. Place the `.jar` file in `/.minecraft/mods/`  
4. Launch Minecraft
